String Manipulations

String manipulations problems appear often in competitive programming, and thus, mastering these will be a big step towards mastering USACO Bronze. String manipulation problems involve using many operations on the strings in the problem, such as concatenating (adding strings onto strings), deletion, or swapping.


First, let's review some basic string operations. Assume we already have the following code:
string str = "hello;"

We can find the char at a certain index (1 in this example) of our string: char c = str[1];
We can add to our string (concatenation): str += "something";
