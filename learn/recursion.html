What is "recursion"?

In plain English, it is the repeated application of a procedure. In computer science, it is when a function, method, or idea calls itself.


Examples

There are many examples of recursion, one famous one is the Fibonacci sequence. Let's say we have a function fib(x).
We have fib(0) = fib(1) = 1 , and since the nth fibonacci number is the sum of the two numbers before it, we have fib(n) = fib(n-1) + fib(n-2). Thus, we can define our simple function[1]:

fib(x):
    if (x is 0 or 1)
        return 1
    else
        return fib(x-1) + fib(x-2)
