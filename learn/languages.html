Languages

Choosing your primary language is a critical step in your computer science career. Each language has its own advantages and disadvantages.

Some of the most popular languages are: C++, Java, Python, and Pascal. In fact, these are all good for USACO! However, because Java, Python, and Pascal compile and run slower compared to C++, it is a good idea to use C++ for USACO. Also, C++ has a lot more data structures compared to the other languages, and is definitely the best language to use regarding algorithmic coding. Plus, if you are looking to compete in IOI (International Olympiad in Informatics), which is the most prestigious competitive progamming competition, C++ is definitely the language to go because only C++ is allowed at IOI.



Because this website isn't a textbook, it moves rather fast, and it is good to get basic knowledge of your language. There are many books available and you can either read them online or buy / rent them. Here is a list of good resources:

C++:
  • The C++ Standard Library: A Tutorial and Reference (2nd Edition) - Nicolai M. Josuttis
  • C++ Primer Plus (6th Edition) (Developer's Library) - Stephen Prata
  • Jumping into C++ - Alex Allain
Java:
  • Java: A Beginner's Guide - Herbert Schildt
  • Java: An Introduction to Problem Solving and Programming (7th Edition) - Walter Savitch
  • Introduction to Java Programming, Comprehensive Version (9th Edition) - Y. Daniel Liang
Python:
  • Python Programming: An Introduction to Computer Science, 2nd Edition - John Zelle
  • Learning Python, 5th Edition - Mark Lutz
  • Think Python - Allen B. Downey
Pascal:
  • Pascal Programming and Problem Solving (4th Edition) - Sanford Leestma & Larry Nyhoff
